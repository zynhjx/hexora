# Hexora Requirements

## Product Overview
Hexora is a beginner-friendly cybersecurity awareness platform designed as a short, interactive experience for events such as school expos. It avoids long technical lessons and instead teaches online safety through quick mini-games.

## Target Audience
- Students and expo participants
- Beginners with little to no cybersecurity background

## Core Learning Goals
- Recognize phishing attempts
- Understand weak vs strong password practices
- Identify unsafe websites and suspicious online behavior

## Main App Flow
1. Authentication
2. Home

Home should include:
- Play
- Leaderboard

## Game Modes (Current Proposal)
- Jumbled letters/words (with hint): user guesses the correct cybersecurity-related word
- 4 Pics 1 Word
- Spot the Threat

## Gameplay Logic Requirements
- Games unlock as user level increases
- Initial state: only 1 game is available
- Each game includes a timer
- Points vary depending on performance and/or time
- More correct answers give more points and EXP

## Progression and Ranking
- EXP is used for level progression
- Points are used for leaderboard ranking

## Event/Reward Context
- Participants enter via a ticket-based system
- Top rank proposal reward: #1 receives PHP 1,000

## Platform Goal
Promote safer online behavior in a fun, simple, and accessible way.
